## 作成

```
yarn create nuxt-app プロジェクト名
```

## プラグイン

### Pug

```
npm i pug pug-loader pug-plain-loader
```

### Sass

```
yarn add npm install sass-loader node-sass
```

```js
// pakage.json

"dependencies": {
    "nuxt": "^2.0.0",
    "node-sass": "^4.12.0",
    "sass-loader": "^7.1.0"
  },

```

### Vuetify

```
yarn add @nuxtjs/vuetify
```

```js
// nuxt.js

  devModules: ['@nuxtjs/vuetify'],
  vuetify: {
      theme: {
        dark: false
      }
    },
  ```

### グローバル CSS

```js
// nuxt.config.js

/*
** Global CSS
*/
css: [
  { src: '~/assets/sass/app.scss', lang: 'scss' },
],
```

### FontAwesome (CDN)
