# vimrc

```
"行番号を採番
set number

"反転でクリップボードにコピー
set clipboard+=unnamed,autoselect

"シンタックスハイライトを有効にする
syntax on
```

ヤンクでもコピーされる！
[vimのヤンク時にMacのクリップボードにもコピーする方法](https://yuu.1000quu.com/vim_yank_to_clipboard)
