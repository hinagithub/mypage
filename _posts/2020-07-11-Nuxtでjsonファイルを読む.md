## プロジェクト内に配置した json を読む方法

![image](https://user-images.githubusercontent.com/44778704/87218178-4d1f7880-c38b-11ea-818c-f5081ac1ddb6.png)

Path

```
assets/json/articles.json
```

中身

```json
[
  {
    "date": "2020.05.29",
    "category": "日常",
    "categoryId": "0",
    "title": "今日は雨だった"
  },
  {
    "date": "2020.05.29",
    "category": "日常",
    "categoryId": "1",
    "title": "Twitterを更新した"
  },
  {
    "date": "2020.05.29",
    "category": "勉強",
    "categoryId": "2",
    "title": "基本情報技術者試験を受けた"
  }
]
```

page/test2.vue

```html
<template lang="pug">
  div
    v-list-item(
      ma-0
      v-for="(article, i) in articles"
      :key="i"
      )
      v-row
        v-col
          v-list-item-subtitle
            span {{ article.date }}
            span |
            span {{ article.category }}
          v-col(cols="12")
          v-list-item-title(ma-0)
            .{{ article.title }}
          v-divider(color="#E0E0E0")
</template>

<script>
  import articles from "@/assets/json/articles.json"; // ここでjsonを読見込む

  export default {
    name: "test2",
    data: () => ({
      articles: articles, // ここに定義する
    }),
  };
</script>
```
